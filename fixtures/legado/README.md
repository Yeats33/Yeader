# Legado Fixtures

Store compatibility fixtures in this directory. Treat every file here as a protocol contract for import/export behavior.

- `sources/`: book source samples
- `rss/`: subscription source samples
- `replace_rules/`: replace rule samples
- `read_configs/`: reading config and theme samples
- `backups/`: backup archives or normalized backup JSON
- `local_books/`: representative `TXT`/`EPUB` files and extracted metadata
- `protocol/`: `legado://` protocol samples and expected parse outputs
